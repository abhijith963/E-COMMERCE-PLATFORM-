# DECOOPX Ecommerce Platform

DECOOPX is a modern, front-end ecommerce platform built with HTML, CSS, and JavaScript. It focuses on clean design, smooth interactions, and a realistic shopping experience. The project demonstrates how a complete ecommerce interface can be built using only front-end technologies.

---

## Overview

This project offers a full shopping experience on the client side. Users can browse different categories, search and filter products, add items to their cart or wishlist, and go through a simple checkout flow. The design is mobile-first and adapts to any screen size, ensuring a consistent user experience across devices.

---

## Features

### Shopping Experience

* Multiple product categories including Electronics, Clothing, Accessories, and Home & Garden
* High-quality images from Unsplash
* Sorting and filtering by price and rating
* Product detail modals showing stock, shipping, and return information
* Wishlist feature to save favorite items

### Cart and Checkout

* Fully functional shopping cart with add, remove, and quantity update features
* Checkout form with validation
* Order summary before confirming purchase
* Toast-style feedback for user actions

### Responsive Design

* Mobile-first layout using CSS Flexbox and Grid
* Adaptive navigation with a hamburger menu on smaller screens
* Layout adjusts smoothly across all devices

### Design and User Experience

* Dark theme with clean, modern accents
* Hover effects and subtle transitions for a polished feel
* Loading indicators and feedback states
* Keyboard accessibility and focus visibility

## How to Use

1. Download or clone the repository
2. Open `index.html` in a browser
3. Browse categories, search for products, and explore product details
4. Add items to the cart or wishlist
5. Proceed through the checkout steps
